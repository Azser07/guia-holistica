# Guía de Sistemas Clínicos Holísticos

Mapeo biológico interactivo con protocolos herbales, preparación y restricciones para seis ejes clínicos funcionales: nervioso, cardiovascular, respiratorio, digestivo, endócrino e inmune.

## Stack

- **React 18** con componentes funcionales y hooks
- **TypeScript** strict mode
- **Vite 5** como bundler
- **CSS custom properties** para sistema de diseño

## Desarrollo

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy

Conecta el repositorio a [Vercel](https://vercel.com). El `vercel.json` detecta Vite automáticamente.
